# 在线聊天系统 - 产品需求文档 (PRD)

> **选题**: 在线聊天系统（85 分 + 语音聊天 5 分）

---

## 一、项目概述

### 1.1 项目目标

构建一个功能完整的在线聊天系统，支持用户注册登录、私聊与群聊、好友分组管理、聊天记录查询与下载、语音聊天等核心功能。

### 1.2 用户角色

| 角色 | 说明 |
|------|------|
| **普通用户** | 注册、登录后使用聊天、好友管理等功能 |
| **管理员** | 管理用户账号、系统配置、内容审核 |

---

## 二、功能需求

### 2.1 用户模块（登录与注册）

#### 2.1.1 注册
- 用户名、密码、确认密码、昵称、头像（可选）
- 用户名唯一性校验
- 密码强度要求（≥6 位，含字母和数字）

#### 2.1.2 登录
- 用户名 + 密码登录
- 记住登录状态
- 登录失败提示（用户名不存在 / 密码错误）

#### 2.1.3 个人信息管理
- 修改昵称、头像、个性签名
- 修改密码（需验证旧密码）
- 在线状态管理（在线、离线、忙碌、勿扰）

---

### 2.2 聊天管理（核心功能）

#### 2.2.1 私聊
- 一对一实时聊天
- 发送文字消息、表情、图片、文件
- 消息已读 / 未读状态
- 历史消息加载（分页、滚动加载）

#### 2.2.2 群聊
- 创建群聊（设置群名、群头像、群公告）
- 邀请好友加入群聊
- 群成员管理（踢人、设为管理员、转让群主）
- 群公告编辑与查看
- 群消息免打扰设置

#### 2.2.3 聊天记录
- 查询聊天记录（支持按关键词搜索）
- 按时间范围筛选聊天记录
- **下载聊天记录到本地**（导出为 TXT / CSV 格式）

#### 2.2.4 语音聊天（+5 分加分项）
- 语音消息录制与发送（录制 → 上传 → 播放）
- ~~一对一实时语音通话~~（技术方案决定不做，复杂度过高）
- 语音消息时长显示
- 语音消息播放控制（暂停、进度条）

---

### 2.3 好友管理

#### 2.3.1 好友操作
- 搜索用户（用户名 / 昵称）
- 发送好友申请（附带验证信息）
- 接受 / 拒绝好友申请
- 删除好友
- 好友资料查看

#### 2.3.2 好友分组
- 默认分组：我的好友、黑名单
- 自定义分组（创建、重命名、删除分组）
- **移动好友到指定分组**
- **删除好友后重新发送验证信息**
- 分组折叠 / 展开显示

#### 2.3.3 好友状态
- 实时在线状态同步
- 好友上线 / 下线通知（可选）

---

### 2.4 管理员模块

#### 2.4.1 用户管理
- 用户列表查看（分页、搜索）
- 禁用 / 启用用户账号
- 重置用户密码

#### 2.4.2 群聊管理
- 查看所有群聊列表
- 解散违规群聊

#### 2.4.3 系统管理
- 系统公告发布
- 敏感词过滤配置

---

## 三、非功能需求

### 3.1 性能需求
- 消息发送延迟 < 500ms
- 支持 100+ 用户同时在线
- 连接异常时自动重连

### 3.2 安全需求
- 密码加密存储
- 用户认证与鉴权
- 通信链路加密
- XSS / SQL 注入防护
- 文件上传类型与大小限制

### 3.3 可用性需求
- 响应式设计，适配 PC 和移动端
- 消息未读数角标提醒
- 消息声音提醒

## 四、核心业务流程

### 4.1 注册登录流程

```
用户 → 填写注册信息 → 校验通过 → 注册成功 → 登录 → 进入主界面
                ↓ 校验失败
            提示错误原因 → 重新填写
```

### 4.2 好友添加流程

```
搜索用户 → 发送好友申请（附验证信息）
    → 对方收到申请通知
    → 接受 → 双方成为好友，进入指定分组
    → 拒绝 → 申请被驳回
```

### 4.3 私聊流程

```
选择好友 → 打开聊天窗口 → 加载历史消息
    → 发送消息 → 实时送达对方
    → 对方不在线 → 上线后拉取离线消息
```

### 4.4 群聊流程

```
创建群聊 → 设置群信息 → 邀请好友入群
    → 群内发送消息 → 所有群成员实时接收
    → 群主/管理员管理群成员
```

### 4.5 聊天记录导出流程

```
打开聊天窗口 → 查看聊天记录
    → 按关键词 / 时间范围筛选
    → 点击导出 → 选择格式（TXT/CSV）→ 下载到本地
```

### 4.6 语音消息流程

```
长按录音 → 录制完成 → 发送语音消息
    → 对方收到语音消息 → 点击播放
```

## 五、数据实体概要

### 5.1 用户
- 用户名、密码、昵称、头像、个性签名
- 在线状态、角色（普通用户 / 管理员）
- 注册时间、最后登录时间

### 5.2 好友关系
- 申请人、被申请人、状态（待验证 / 已通过 / 已拒绝）
- 所属分组、备注名、验证信息
- 建立时间

### 5.3 好友分组
- 分组名称、排序序号
- 是否默认分组
- 所属用户

### 5.4 私聊消息
- 发送者、接收者、消息内容
- 消息类型（文字 / 图片 / 文件 / 语音）
- 已读状态、发送时间

### 5.5 群聊
- 群名称、群头像、群公告
- 群主、创建时间

### 5.6 群成员
- 所属群、用户、角色（群主 / 管理员 / 成员）
- 是否免打扰、加入时间

### 5.7 群聊消息
- 所属群、发送者、消息内容
- 消息类型（文字 / 图片 / 文件 / 语音）
- 发送时间

### 5.8 语音消息
- 关联消息、文件地址
- 语音时长、文件大小
